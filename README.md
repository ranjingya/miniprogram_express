# 校园快递超市微信小程序  
## 后端接口实现：  
[校园快递超市微信小程序后端实现](https://github.com/ranjingya/miniprogram_express_backend)  

## 项目概述  

本次微信小程序项目由小组共同完成。  
本人负责首页页面、快递查询、购物车页面及功能、以及后端接口的实现  

### 实现功能  

首页快递在线查询、个人中心token登录、快递查询记录保存及模糊搜索、  
查询记录页面数据懒加载、商城中心、商品分类加载、购物车、地址簿、  
等等
  
### 开发环境  
系统：Windows 11  
IDE：微信开发者工具、VScode
